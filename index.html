About the Project

This project demonstrates my growing knowledge in HTML, CSS, and web development. It is part of my Creative Computing coursework at SMU, where I am exploring interactive and aesthetic website design.

The focus of this site is to serve as a landing page for my ongoing development work. It will later expand to include functionality, design improvements, and deeper interactive features using JavaScript and back-end technologies.

In this iteration, I have built a simple yet organized layout using semantic HTML structure and external CSS styling.
