About the Project

This project demonstrates my growing knowledge in HTML, CSS, and web development. It’s part of my Creative Computing coursework at SMU, where I’m exploring interactive and aesthetic website design.

The focus of this site is to serve as a landing page for my ongoing development work. It will later expand to include functionality, design improvements, and deeper interactive features using JavaScript and back-end technologies.

In this iteration, I’ve built a simple yet organized layout using semantic HTML structure and external CSS styling. The design reflects both clarity and creativity — keeping things readable, modern, and easy to navigate.
